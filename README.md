# 明日背包

一个给学生整理明天书包的轻量网页工具。

## 这是什么

这个项目是第一版 MVP，重点是让使用者在晚上快速确认明天要带的东西。

目前支援：

- 星期一到星期五固定课程表
- 为每个科目设定固定物品
- 首页自动显示下一次上课日要带的东西
- 同一天重复物品自动合并
- 为当天新增临时项目
- 勾选完成状态保存在当前浏览器

## 项目结构

- `index.html`：页面结构
- `style.css`：画面样式
- `app.js`：互动逻辑与本机资料保存

## 本机预览

直接用浏览器打开 `index.html` 即可使用。

## GitHub Pages

这个项目是纯静态网页，很适合部署到 GitHub Pages。

发布后可直接通过链接打开，不需要额外安装。

## 目前的资料保存方式

第一版使用浏览器本机储存。

这代表：

- 每个装置会有自己独立的一份资料
- 不同同学之间不会自动同步
- 清除浏览器资料后，设定内容可能会消失
